# AssetFlow Frontend

AssetFlow is a comprehensive asset management system designed to track, manage, and audit organizational assets efficiently. This repository contains the frontend portion of the application.

## 🚀 Features

- **Asset Dashboard**: High-level overview of total assets, their status, and recent activities.
- **Asset Registration**: Easy onboarding of new assets into the system.
- **Asset Allocation & Return**: Track who currently has which asset and streamline the return process.
- **Asset Directory**: A detailed list and search interface for all company assets.
- **Auditing**: Dedicated section to conduct and log asset audits.
- **Maintenance Tracking**: Keep a history of maintenance schedules and repairs.
- **User & Department Management**: Organize employees and departments for accurate asset assignment.

## 📂 Project Structure

This project uses vanilla HTML, CSS, and JavaScript. 

- `index.html` - The main entry point (Login/Dashboard).
- `css/` - Contains the stylesheets (`main.css`, `components.css`, `sidebar.css`, etc.)
- `js/` - Contains all logic and API integration (`api.js`, `dashboard.js`, `asset-registration.js`, etc.)
- `assets/`, `audit/`, `booking/`, `organization/` - Specific modules/pages of the application.

## 🛠️ Setup & Installation

Since this project uses plain HTML/JS/CSS, there is no build step required. 

1. Clone the repository.
2. Open `index.html` in your web browser, or use a local development server like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code for a better experience.
3. Make sure the **AssetFlow Backend** is running so that the frontend can fetch the data from the API endpoints.
